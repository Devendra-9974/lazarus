# 🏥 Lazarus Project

A full-stack web application for managing patient data using **React (Frontend)** and **Spring Boot (Backend)**.

---

## 🚀 Live Demo

- 🌐 Frontend: https://lazfront.vercel.app/
- ⚙️ Backend: https://lazarus-2.onrender.com/api/patients

---

## 📌 Overview

The Lazarus Project allows users to manage patient records efficiently.  
It provides a clean UI for interacting with data and a powerful backend API for handling operations.

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vercel (Deployment)

### Backend
- Java Spring Boot
- REST API
- Render (Deployment)
